# ⚡ Simple Workflow: Single Blog Post

**Purpose:** Ship a focused blog post quickly using a single specialist.  
**Duration:** 30–60 minutes.  
**Specialists:** Copywriter (examples/specialists/completed-copywriter.yaml).

## When to Use This
- Topic and audience are already defined.  
- You need publishing speed more than strategic depth.  
- No research, cross-functional handoff, or go-to-market coordination required.

## Workflow Overview

### Step 1 — Direct Activation
```
Acting as the Copywriter from my AI-Staff-HQ, write a 600-word blog post for [audience] about [topic].
Tone: [tone]
Key points to cover: [bullet list]
Call to action: [desired action]
```

### Step 2 — Rapid Review & Iteration
- Read the draft immediately.  
- Provide one round of focused feedback (e.g., add proof point, tighten intro).  
- Re-activate the Copywriter with the requested changes.  

### Step 3 — Finalize & Publish
- Paste the approved draft into your CMS.  
- Log the final prompt + output in your `knowledge-base/` for future reuse.  
- Note any insights that should feed into a broader content plan.

## Why This Works
- Zero coordination overhead; Copywriter has all required context.  
- Direct feedback loop keeps the iteration tight.  
- Works entirely within a single context window, preserving efficiency.

## When to Upgrade to a Complex Workflow
- Need keyword strategy or competitive analysis → involve the Market Analyst & SEO Specialist.  
- Part of a multi-asset campaign → run the brand development or content pipeline workflow.  
- Requires visual assets → pair with the Art Director and use a coordinated workflow.
