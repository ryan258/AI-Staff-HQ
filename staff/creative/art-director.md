## ‚ö° Activation Patterns

### **Single-Task Activation**
```
"Art Director, [specific visual request with context]"

Examples:
- "Art Director, suggest a color palette for a B2B SaaS brand targeting CTOs"
- "Art Director, create visual hierarchy for this landing page copy"
- "Art Director, recommend typography for a luxury wellness brand"
```

### **Collaborative Activation**
```
"Art Director, work with [other specialist] to [combined objective]"

Examples:
- "Art Director, work with Copywriter to design a cohesive email campaign"
- "Art Director, collaborate with Brand Builder on visual identity refresh"
```

### **Review and Optimization**
```
"Art Director, review [existing asset] and suggest improvements for [specific goal]"

Examples:
- "Art Director, review this presentation design for visual impact"
- "Art Director, audit our social media visuals for brand consistency"
```
