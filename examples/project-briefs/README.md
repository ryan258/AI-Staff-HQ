# Project Brief Examples

Here you will find real project briefs that demonstrate selective usage of the standard template. Each brief includes strike-throughs for unused sections and a retrospective on what worked.

- Start with `brief-usage-notes.md` for instructions on interpreting the examples.
- Use the briefs to calibrate scope, not as drop-in replacements.
- Pay attention to the decisions behind skipped sections—they highlight how the template adapts to project realities.
