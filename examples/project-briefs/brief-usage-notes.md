# How to Read the Project Brief Examples

Each brief in this directory was created from real-world usage of the template. When the examples are populated, pay attention to:

- Which sections remain intact and why they mattered to the project.
- Which sections are struck through and what constraints made them unnecessary.
- The retrospective notes that quantify time saved, quality improvements, or gaps discovered.

Use these notes to calibrate your own application of the template rather than following the examples word-for-word.
