# AI-Staff-HQ Examples

These examples show how a personal AI workforce operates in practice. They illustrate completed specialists, real project briefs, actual workflow transcripts, and before/after comparisons drawn from live work. 

> These are reference examples, not templates to copy. Use them to understand decision patterns, then build your own equivalents.

## Start Here
1. **`specialists/completed-copywriter.yaml`** – See what a production-ready specialist looks like.  
2. **`project-briefs/simple-blog-post-brief.md`** – Watch how the project brief template is used selectively.  
3. **`workflows/blog-post-execution-transcript.md`** – Follow a complete execution log with real prompts.  
4. **`before-after/prompt-vs-workflow-comparison.md`** – Understand the ROI difference between one-shot prompting and systematic workflows.

## Directory Overview
- `specialists/` – Six completed specialists with notes on creation and integration trade-offs.
- `project-briefs/` – Three project briefs (simple, medium, complex) with retrospectives and selective template usage.
- `workflows/` – Two execution transcripts plus a single-specialist workflow for quick wins.
- `before-after/` – Comparisons that quantify the value of systematic workflows against one-shot prompting.

Each subdirectory includes its own README with usage guidance and notes on how the examples were produced.
