# Before/After Comparisons

This directory contrasts one-shot prompting with structured AI-Staff-HQ workflows. Each comparison highlights where systematic orchestration pays off and where the overhead might be unnecessary.

- Start with `prompt-vs-workflow-comparison.md` for a narrative comparison of a single scenario.
- Use `roi-analysis.md` to quantify time, quality, and reusability trade-offs.
- Expect candid notes on when the heavier process is overkill—the aim is honest evaluation, not marketing.
